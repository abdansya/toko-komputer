## Install

- Clone from git repository
```
git clone https://github.com/abdansya/toko-komputer.git
```
- Copy .env.example to .env and adjust the value
- Composer install
```
composer install
```
- Run migration and seeder
```
php artisan migrate --seed
```
- Run server in localhost
```
php artisan serve
```
